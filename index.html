Attribute selectors

Selector name Selector syntax Selector example Selector description Example description
Attribute selector [attr] {
style properties
}		 [autoplay] {
color: green;
}		 Represents elements with an attribute name of attr. Selects all elements with an autoplay attribute.
[attr=value] {
style properties
}		 [type="checkbox"] {
background-color: skyblue;
}		 Represents elements with an attribute name of attr whose value is exactly value. Selects all elements with a type attribute equal to checkbox.
[attr~=value] {
style properties
}		 [lang~="en-us"] {
margin: 2em;
}		 Represents elements with an attribute name of attr whose value is a whitespace-separated list of words, one of which is exactly value. Selects all elements with a lang attribute containing en-us.
[attr|=value] {
style properties
}		 [lang|="zh"] {
background-color: skyblue;
}		 Represents elements with an attribute name of attr whose value can be exactly value or can begin with value immediately followed by a hyphen, - (U+002D). It is often used for language subcode matches. Selects all elements with a lang attribute matching exactly or starting with zh e.g. zh-CN, zh-TW.
[attr^=value] {
style properties
}		 [href^="#"] {
background-color: skyblue;
}		 Represents elements with an attribute name of attr whose value is prefixed (preceded) by value. Selects all elements with a href attribute which value starts with #.
[attr$=value] {
style properties
}		 [href$=".org"] {
background-color: skyblue;
}		 Represents elements with an attribute name of attr whose value is suffixed (followed) by value. Selects all elements with a href attribute which value ends with .org.
[attr*=value] {
style properties
}		 [href*="example"] {
background-color: skyblue;
}		 Represents elements with an attribute name of attr whose value contains at least one occurrence of value within the string. Selects all elements with a href attribute which value contains example anywhere in the string.
[attr operator value i] {
style properties
}		 [href*="insensitive" i] {
background-color: skyblue;
}		 Adding an i (or I) before the closing bracket causes the value to be compared case-insensitively (for characters within the ASCII range). Selects all elements with a href attribute which value contains insensitive anywhere in the string, regardless of capitalization.
